<h1><strong>ANGULAR TIMER&nbsp; </strong></h1>
<p style="text-align: justify;">This project is a timer application that is made using Angular and Electron with the use of the following web tools: HTML, CSS, and JavaScript.&nbsp;</p>
<p style="text-align: justify;"><strong>Requirements:</strong></p>
<ul>
  <li style="text-align: justify;">Nodejs</li>
  <li style="text-align: justify;">Code Editor (e.g Visual Studio Code, Atom)</li>
</ul>
<p style="text-align: justify;"><strong>Setup Instructions</strong>:</p>
<pre><span class="pl-c"># Clone this repository</span>
<strong><em>git clone https://github.com/NKVitug/Angular-Timer.git</em>
</strong><span class="pl-c"># Go into the repository</span>
<strong><em><span class="pl-c1">cd</span> Angular-Timer</em>
</strong><span class="pl-c"># Install dependencies</span>
<strong><em>npm install -g @angular/cli</em>
<strong><em>npm install electron --save-dev</em>
</strong><span class="pl-c"># Build the app</span>
<strong><em>npm run electron-build</em></strong></pre>
